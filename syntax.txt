Синтаксис языка состоит из следующих команд:

setchar char c - устанавливает значение в текущей ячейке на c
walk int n - перейти на n ячеек
change int v - изменить значение в текущей ячейке на v
mov int n - переместить значение текущей ячейки на n клеток
copy int n int t - копировать значение текущей ячейки в ячейку через n клеток с использованием буффера,
расположенного на t клеток от текущей
print unsigned int n - вывести значения ячеек от текущей до следующей через n клеток
split string v - разбить строку v на символы и распределить их по следующим ячейкам
alias name value - создать "переменную" с именем name и значением value
@name - получить значение из переменной с именем name

Каждый вызов команды заканчивается точкой с запятой.

На самом деле это не совсем переменные, так как они используются только во время трансляции
